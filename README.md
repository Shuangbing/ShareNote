# ShareNote
- A Web App for Share notes.

# Usage

1. ".env.example"を".env"にコピーしデータベース情報を変更する
2. composer installを実行しライブラリをインストールする
3. php artisan migrateを実行しテーブルを作成する
4. php artisan serveを実行し開発サーバを起動する
